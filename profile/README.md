## Korean

### 핵심 포인트

- `guide_image.md` 기반 이미지 프롬프트 템플릿
- `guide_vedio.md` 기반 영상 프롬프트 템플릿
- `guide_plan.md` 기반 서비스 기획 요약 반영
- 이미지 생성, 업로드, 영상 생성 흐름 통합

### 빠른 사용 순서

1. 이미지 탭에서 한글 프롬프트 입력
2. 템플릿으로 주체, 분위기, 배경, 조명, 디테일 보완
3. 자동 확장 프롬프트 확인 후 이미지 생성
4. 생성 이미지 또는 업로드 이미지 선택
5. 영상 탭에서 카메라/객체/환경/톤 입력
6. 자동 확장 영상 프롬프트 확인 후 영상 생성

### 실행

```bash
cd C:\Users\ggg99\Desktop\ai\all-in-one-site
npm install
npm run dev
```

`http://localhost:3000`

## English

### Highlights

- Guided image prompting from `guide_image.md`
- Guided video prompting from `guide_vedio.md`
- Product-planning summary from `guide_plan.md`
- Unified image, upload, and video flow

### Quick Flow

1. Enter a Korean prompt in the image tab
2. Fill the structured template
3. Review the expanded prompt and generate images
4. Select a generated image or upload your own
5. Define video motion and atmosphere
6. Review the expanded video prompt and generate the result

### Run

```bash
cd C:\Users\ggg99\Desktop\ai\all-in-one-site
npm install
npm run dev
```

`http://localhost:3000`
